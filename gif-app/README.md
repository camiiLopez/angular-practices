# GifsApp


1. Clonar el repositorio
2. Instalar las dependencias con `npm install`
3. Crear un archivo environment con la siguiente plantilla:
    ```
    export const environment = {
    production: false,
    companyName: 'Gifs',
    companyName2: 'App',
    companySlogan: 'Maneja tus gifs',

    // ApiKeys
    giphyApiKey : '',
    // URLS
    giphyUrl: ''
    };
    ```
4. Iniciar el servidor con `ng serve`
5. Navegar a `http://localhost:4200/`
